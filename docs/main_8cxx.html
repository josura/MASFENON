Detailed Description

Main function for the MASFENON simulation framework. Without the use of MPI.

This file contains the main function for running the MASFENON simulation framework without MPI. It handles command-line arguments, initializes the computation, and executes the simulation.

Note
This version of the main function is designed non-MPI platforms.
Warning
This version is deprecated and behind the MPI version. It is kept for backward compatibility and testing purposes.
