Detailed Description

Simple logging utility for writing formatted messages to an output stream.

The Logger class allows writing generic log messages, warnings, and errors to a provided output stream. Logging can be toggled on or off.

Delete copy constructor and assignment operator to prevent copying.

This ensures that the Logger instance is unique and cannot be copied, which is important for thread safety and resource management.

Note
Copying a Logger instance could lead to issues with concurrent access to the same output stream.
