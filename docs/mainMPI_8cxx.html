Detailed Description

Main function for the MASFENON simulation framework. With the use of MPI.

This file contains the main function for running the MASFENON simulation framework with MPI. It handles command-line arguments, initializes the computation, and executes the simulation.

Note
This version of the main function is designed for MPI platforms.
This version is the most up to date and is the one that should be used for the simulations.
