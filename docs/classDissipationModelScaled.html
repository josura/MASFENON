Detailed Description

Class for managing scaled dissipation dynamics in MASFENON.

This class inherits from the DissipationModel class and implements scaled dissipation logic.

The scaled dissipation model uses a scale function to compute the dissipation term.

The dissipation term is computed as the product of the input vector and the scale function.

The scale function can be set and modified as needed.
