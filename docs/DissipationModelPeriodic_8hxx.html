Detailed Description

Defines the DissipationModelPeriodic class used for managing periodic dissipation dynamics for the computation of the perturbation in MASFENON.

The DissipationModelPeriodic class inherits from the DissipationModel class and implements periodic dissipation logic.

The periodic dissipation model uses a set of phases, periods, and amplitudes to compute the dissipation term.

The dissipation term is computed as a sum of sine functions with the given phases, periods, and amplitudes.
