Detailed Description

Defines the Computation class used for managing the logic of intra- and inter-network dynamics in MASFENON.

This class handles core data structures, including graphs and matrices, and executes propagation, dissipation, and conservation steps using models. It supports augmented graphs and integrates with Armadillo for efficient computations.
