Defines the ConservationModelVectorized class for vectorized conservation computations.

The ConservationModelVectorized class provides methods for applying conservation logic to the perturbation computation.

The conservation class uses a scale function to determine the scaling of the conservation term. The scale function can be set and modified as needed.
